#Лаборатория*веб*платформы

Look Ma, это я в 2025 пишу фича-флаги прямо в CSS!

```css
.component {
  /* обычные стили */
  /* ... */

  @container style(--wow-such-a-feature: true) {
    /* стили за фича-флагом */
    /* ... */
  }

  @container style(--wow-such-an-another-feature: true) {
    /* стили за фича-флагом */
    /* ... */
  }
```
