### Новости

- в `@xstate/store@3.4.0` (минималистичный стейт-менеджер под капотом `xstate`, который можно использовать без полного обвеса), [добавили «атомарное» API](https://github.com/statelyai/xstate/releases/tag/%40xstate%2Fstore%403.4.0) `createAtom()` с возможностью создавать атомарные хранилища, комбинировать их друг с другом и со сторами
- в техническом комитете Node.js [решили исключить](https://socket.dev/blog/node-js-tsc-votes-to-stop-distributing-corepack) `corepack` из базовой поставки начиная с Node.js 25+ (в более ранних версиях он останется экспериментальной фичей), теперь нужно будет ставить отдельно `npm install -g corepack`
- библиотека для анимации [Motion портирована на Vue](https://motion.dev/blog/introducing-motion-for-vue), теперь можно во так:

```vue
<template>
  <motion.div
    :initial="{ opacity: 0, scale: 0 }"
    :animate="{ opacity: 1, scale: 1 }"
  />
</template>
```

- [в Next.js найдена уязвимость](https://zhero-web-sec.github.io/research-and-things/nextjs-and-the-corrupt-middleware) (больше похожая на бэкдор), позволяющая обходить мидлвары спец-заголовком `x-middleware-subrequest`, то есть если у вас в мидлваре выполняется что-то важное, например, авторизация, то её сейчас можно скипнуть (если не обновить либу)
- [релизнулся Rsdoctor 1.0](https://rsdoctor.dev/blog/release/release-note-1_0), webpack-совместимый анализатор бандла, аналог тулов  [webpack-bundle-analyzer](https://github.com/webpack-contrib/webpack-bundle-analyzer) и [Statoscope](https://github.com/statoscope/statoscope/): с помощью него можно получить ответы на вопросы «почему билд медленный», «как лучше разбить бандл на чанки», «почему увеличился размер бандла»
- в Chrome 135 будут добавлены новые фишки:
  - [кастомизируемый элемент `<select>`](https://developer.chrome.com/blog/a-customizable-select) с помощью свойства `appearance: base-select`, в который можно добавить произвольный HTML
  - [псевдоэлементы](https://developer.chrome.com/blog/carousels-with-css) `::scroll-button()` и `::scroll-marker()` для создания и стилизации «нативных» каруселей
- [большое обновление Valibot v1.0.0](https://valibot.dev/blog/valibot-v1-the-1-kb-schema-library/) (рантайм-валидатора схем) спустя 15 бет и 5 rc, много добавлений/изменений

### Проекты

- [mise](https://github.com/jdx/mise) — мультитул для переключения версий node, python, ruby (как nvm) + переключения env-переменных (как direnv) + запуска тасков (как make)
- [kaluma](https://github.com/kaluma-project/kaluma) — мини-JS-рантайм для Raspberry Pi (вдумайтесь в параметры устройства: 300KB ROM, 64KB RAM)
- [http-decision-diagram](https://github.com/for-GET/http-decision-diagram) — блок-схема принятия решения, какой HTTP-статус отдать клиенту
- [Parvus](https://github.com/deoostfrees/Parvus) — доступный лайтбокс без зависимостей на нативном `<dialog>`-е с интернализацией

### Статьи и демки

#### JS

- а вы знали, что можно [обратиться к текущему выполняющемуся скрипту](https://leanrada.com/notes/inline-rendering-currentscript/) через  [`document.currentScript`](https://developer.mozilla.org/en-US/docs/Web/API/Document/currentScript)? Это может быть полезно, чтобы, к примеру, динамически заменить скрипт на произвольный контент, получается такой дешёвый движок рантайм-шаблонов для всяких мелочей:

```html
Come on, it’s
<script>
  document.currentScript.replaceWith(new Date().getFullYear());
</script>
```

#### CSS

- [ещё один юзкейс](https://css-tip.com/custom-progress/) для нового типизированного `attr()`: прокидывание значений атрибутов `value` и `max` элемента `<progress>` напрямую в CSS
- `mask` + `repeating-conic-gradient` + немного тригонометрии = [зигзаг](https://css-tip.com/zig-zag-edge/)
- если нужно что-то по-быстрому поправить в загруженном сайте без инспектора в дев-тулзах, можно использовать [дизайн-режим](https://www.smashingmagazine.com/2025/03/previewing-content-changes-work-documentdesignmode/) (ввести `document.designMode = "on"` в консоли)
- а вот и [миксины подвозят в канарейку Chrome](https://nerdy.dev/css-mixins-ready-for-experimentation), как обычно подождём 3 года и можно будет пользовать:

```css
@mixin --box {
  aspect-ratio: 1;
  inline-size: 100px;
  block-size: 100px;
}

.box {
  @apply --box;
}
```

#### HTML

- полезные фичи `<fieldset>` [при создании форм](https://html-css-tip-of-the-week.netlify.app/tip/fieldset-and-legend/): можно дизейблить целиком всю группу полей через атрибут `disabled` у `<fieldset>`, а также `<fieldset>` может находиться отдельно от `<form>` и быть связанным с формой атрибутом `form` со значением `id` формы
