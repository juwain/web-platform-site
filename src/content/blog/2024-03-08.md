**#Пульс*веб*платформы 08.03.2024**

**Новости**

- [вышел TypeScript 5.4:](https://devblogs.microsoft.com/typescript/announcing-typescript-5-4/) нового с RC версии почти ничего не подвезли, но там было много всего хорошего, можно обновляться
- [вышел Parcel v2.12.0:](https://parceljs.org/blog/v2-12-0/) теперь умеет прекомпилировать JS-функции в билд-тайме вместо включения в общий бандл (вдохновлено макросами из Bun); на сайте появился [REPL](https://repl.parceljs.org/) (интерактивный редактор с отображением результата); теперь под капотом Lightning CSS бандлит CSS-файлы; возможность вручную сконфигурировать [shared-бандлы](https://parceljs.org/features/code-splitting/#shared-bundles)
- [вышел Babel 7.24.0:](https://babeljs.io/blog/2024/02/28/7.24.0) теперь поддерживает JSON modules imports, а также последнюю актуальную версию декораторов
- в [Safari 17.4 появится поддержка](https://webkit.org/blog/15054/an-html-switch-control/) нового «пропатченного» чекбокса — переключателя (`<input type=checkbox switch checked>` с псевдоэлементами ::thumb и ::track), также [появились](https://webkit.org/blog/15063/webkit-features-in-safari-17-4/) вертикальные контролы форм, `<hr>` в `<select>`, обновление движка Inline Layout, работающий align-content для таблиц и блоков (не только гридов и флексбоксов), `@scope` и `:scope`, `Promise.withResolvers`, `Object.groupBy` и `Map.groupBy` и [кое-что ещё](https://webkit.org/blog/15063/webkit-features-in-safari-17-4/)
- [вышла PixiJS v8:](https://pixijs.com/blog/pixi-v8-launches) либа для 2d анимации с десятилетней историей хорошо бустанулась по перфу, подружилась с WebGPU, изменила структуру пакета, получила обновления в графических, векторных и текстовых API
- [вышла бета Million Lint:](https://million.dev/blog/lint) интересный концепт линтера перфоманса React-приложения, который в реалтайме показывает в коде на манер eslint количество ререндеров и время
- React Compiler aka Forget (автомемоизатор) [не будет включён в релиз React 19](https://twitter.com/en_JS/status/1760433969118769654)

**Проекты**

- [readability](https://github.com/mozilla/readability) — JS-версия браузерного режима для чтения из Firefox: это когда остаётся только текст, а стили и оформление с баннерами уходит
- [embla-carousel](https://github.com/davidjerleke/embla-carousel) — минималистичная JS-карусель, без зависимостей, для доп функциональности подключаются плагины
- [pkgx](https://pkgx.sh/) — запускальщик _всего_ здесь и сейчас без установки <s>и смс</s>, что не умеет npx, например, мождно запустить ноду pkgx node@16 или бан pkgx bun run
- [turbo repo](https://turbo.build/repo/) — тула для организации более быстрой работы команд запуска и сборки в монорепах (yarn/npm/pnpm устанавливает пакеты, turbo запускает с кешированием)
- [capo.js](https://rviscomi.github.io/capo.js/) — инструмент для анализа правильности в порядке тегов в `<head>`: покажет, в какой последовательности расставить теги будет правильнее

**Статьи и демки**

**JS**

- 4 издание прекрасной книги про JS — [Eloquent JavaScript](https://eloquentjavascript.net/) — для программистов любых возрастов и вероисповедания
- [первые отзывы о JSR](https://socket.dev/blog/jsr-new-javascript-package-registry) — альтернативному npm пакетному менеджеру от создателей deno: пока очень нишево, привязано к deno и немного чем отличается (TS-first, only ESM-модули), но, тем не менее, конкуренция — всегда хорошо
- если вы, как и [автор статьи,](https://shubhamjain.co/2024/02/29/why-is-number-package-have-59m-downloads/) удивляетесь, почему у однострочных пакетов типа is-number столько скачиваний на npm, то бывает это сам автор раздувает использование своей либы через другие пакеты (naturally, пакеты с пакетами)
- как на Performance API собрать (на коленке) [основные пользовательские метрики перфа:](https://www.smashingmagazine.com/2024/02/reporting-core-web-vitals-performance-api/) Largest Contentful Paint, First Contentful Paint, Cumulative Layout Shift, Interaction To Next Paint, Long Animation Frames

**CSS**

- реакт реактом, а [переносить текст со строчки на строчку](https://adrianroselli.com/2024/02/techniques-to-break-words.html) всё равно нужно — гайд по word-break, oveflow-wrap, hyphens, <wbr>, &shy (с последней парочкой нужно быть аккуратными)
- [скролл-анимация триггерит](https://ryanmulligan.dev/blog/scroll-triggered-animations-style-queries/) смену кастомного свойства --animate: true, а по нему применяется Style Query @container style(--animate: true), который анимирует появившийся во вьюпорте элемент — самый блидинг из возможных эджей
- хочешь не хочешь, а рано или поздно придёт она — задача на стилизацию печатных страниц, в этом случае пригодится [детальный гайд](https://voussoir.net/writing/css_for_printing) по @page и @media print в разных вариациях
