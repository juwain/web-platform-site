**Новости**

- вышел [Electron 28.0.0](https://www.electronjs.org/blog/electron-28-0): добавлена поддержка ES-модулей и там всё оч интересно: главный процесс Electron-а запускается в ноде и соответственно [использует имплементацию ESM ноды](https://www.electronjs.org/docs/latest/tutorial/esm), а рендерится всё в хромиуме и там уже другая имплементация, Electron выбирает, как быть, в зависимости от контекста
- вышла библиотека [StyleX от фб](https://github.com/facebook/stylex): из плюсов — типизация, генерация в билд-тайме, из минусов — непонятно зачем это нужно сейчас кому-то, кроме фб
- Remix [запланировал реализацию](https://github.com/orgs/remix-run/projects/5/views/1) React Server Components (видимо, вынужденно)
- вышел [Vitest v1.0.0](https://github.com/vitest-dev/vitest/releases/tag/v1.0.0): кажется уже устаканилось настолько, чтобы внедрять (1.0.0 внушает доверие)
- вышел [новый HTML5 Boilerplate](https://htmlcssjavascript.com/web/html5-boilerplate-v9-0-0-released/) (да, это не шутка! внимание, может вызвать ностальгические чувства)

**Проекты**

- как вам стейт-менеджер веб-приложений в виде SQL-бд (чудны дела твои, эволюция)? такое есть в [SQLSync](https://github.com/orbitinghail/sqlsync) (а для выборки из стора можно пользовать SQL, yay)
- симпатичная либа компонентов на тейлвинде [floatui](https://github.com/MarsX-dev/floatui) (проект настолько свежий, что не успели ещё поправить ридми)
- [tsdiagram](https://github.com/3rd/tsdiagram) сгенерит для вас на лету красивую диаграмму из хитросплетений типов
- just another либа для манипуляций с цветом [culori](https://github.com/Evercoder/culori)
- [тул для сбора перфоманс-метрик](https://github.com/Zizzamia/perfume.js) (в том числе Web Vitals) на пользовательских устройствах и отсыла их обратно через google-аналитику либо другие аналитические тулы (главное чтобы браузеры не блочили аналитические тулы)

**Статьи и демки**

**JS**

- как улучшить [перфоманс-метрику INP](https://calendar.perfplanet.com/2023/yielding-main-thread-breaking-up-tasks-fix-inp/): предсказуемые способы разгрузить основной поток `setTimeout`, `requestAnimationFrame`, `requestIdleCallback`, а также неведомый `scheduler.yield()`
- как плохой INP может [уронить бизнес-метрики](https://www.speedcurve.com/blog/INP-user-experience-correlation/)
- [ещё одна попытка](https://gomakethings.com/the-elevator-pitch-for-web-components/) заинтересовать в веб-компонентах: в них можно бесплатно инкапсулировать логику, есть встроенные колбеки на добавление и удаление из DOM-а и изменение «пропсов»
- как [заменить](https://pawelgrzybek.com/i-replaced-npm-yarn-and-nvm-with-pnpm/) npm, yarn и nvm на pnpm (правда не очень понятно зачем)
- [сборник рецептов для Puppeteer](https://proxiesapi.com/articles/the-complete-puppeteer-cheatsheet) (от простого волшебства к сложным заклинаниям)
- как [сделать скриншот фрагмента страницы](https://lirantal.com/blog/advanced-usage-patterns-for-taking-page-element-screenshots-with-playwright/) с помощью Playwright
- old but gold: что такое и с чем едят [JavaScript Import maps](https://spidermonkey.dev/blog/2023/02/23/javascript-import-maps-part-1-introduction.html)
- паттерны проектирования реакт-компонентов: [as и asChild](https://www.bekk.christmas/post/2023/1/polymorphism-in-react) (is this a полиморфизм?)
- история того, как в JS появилось [именно такое ООП](https://www.smashingmagazine.com/2023/12/marketing-changed-oop-javascript/), какое есть сейчас (читать с кофе, в кресле, укрывшись пледом)
- [реактивный JS-фреймворк на коленке](https://nolanlawson.com/2023/12/02/lets-learn-how-modern-javascript-frameworks-work-by-building-one/), откуда можно узнать: юзкейс для `Proxy`, незаслуженно неизвестное API `queueMicrotask`, ликбез по [tagged template literals](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Template_literals#tagged_templates)
- [хитрая магия](https://timjohns.ca/typescripts-hidden-feature-subtypes.html), чтобы делать «подтипы» в TS (после использования сжечь проект!)
